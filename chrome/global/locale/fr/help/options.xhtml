Options de &tabsidebar.brand;

Ce document explique toutes les options disponibles dans la fenêtre d'options de &tabsidebar.brand;.

Dans cette section :
  • Afficher les options
  • Paramètres généraux

Afficher les options

Pour voir les options, allez dans le menu Outils > Extensions. Sélectionnez alors &tabsidebar.brand; et cliquez sur le bouton Options.

Paramètres généraux

&options.hidetabs;
Cette option vous permet de masquer la barre d'onglets principale du navigateur quand le panneau latéral est ouvert. La barre d'onglets sera à nouveau visible lorsque vous fermerez Tab Sidebar.
&options.buttons;
Afficher les boutons de navigation pour chaque onglet. Ces boutons sont similaires à ceux de la barre d'outils, c'est-à-dire stop, actualiser, reculer et avancer.
