&tabsidebar.brand; Options

This document explains all of the options available in the options window of &tabsidebar.brand;.

In this section:
  • Displaying Options
  • General Settings
  • Display Settings
  • Preview Settings

Displaying Options

To view the options go to Tools > Extensions. Then select &tabsidebar.brand; and click the Options button.

General Settings

&paneGeneral.hidetabs.label;:
This option allows you to hide the browser's main tab bar when the tab sidebar is open. The tab bar will be made visible again when you close the tab sidebar.

&panePreviews.autorefresh.label;:
For pages that continually change it may be desirable to have the preview redraw continually. This option allows you to do this and lets you specify how often (in seconds) to draw the preview. The minimum time is every 5 seconds.

Display Settings

The display options control what information is displayed for each tab in the sidebar. The following options are available:

&paneDisplay.content.favicon.label;:
This option controls the display of the website's icon in the display.

&paneDisplay.content.close.label;:
Displays a button letting you close the tab quickly.

&paneDisplay.content.title.label;:
Shows the title of the website displayed in the tab.

&paneDisplay.content.uri.label;:
Shows the address of the address of the website displayed in the tab.

&paneDisplay.content.navigation.label;:
Displays a navigation buttons that can be clicked to reload a page and stop loading.
