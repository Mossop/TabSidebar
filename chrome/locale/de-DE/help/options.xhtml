&tabsidebar.brand; Options

This document explains all of the options available in the options window of &tabsidebar.brand;.

In this section:
  • Displaying Options
  • General Settings
  • Display Settings
  • Preview Settings

Displaying Options

There are two ways to display the options for &tabsidebar.brand;. Firstly you can view them from the extension manager. Do this by going to Tools > Extensions. Then select &tabsidebar.brand; and click the Options button. Secondly you can just right-click Press &ctrlKey;, click on the sidebar and select &tabsidebar.options.label;.

General Settings

&paneGeneral.hidetabs.label;:
This option allows you to hide the browser's main tab bar when the tab sidebar is open. The tab bar will be made visible again when you close the tab sidebar.

Hide Mode

The &tabsidebar.brand; can automatically hide from the window when you do not need it. You may choose one of the following modes to configure this:

&paneGeneral.mode.normal.label;:
In this mode the sidebar never hides. This is the default and is how most normal sidebars display.

&paneGeneral.mode.click.label;:
This mode provides a slim click panel on the left of the window. Clicking this will open and close the sidebar.

&paneGeneral.mode.auto.label;:
The final mode allows the sidebar to autmoatically hide. Simply move your mouse over the hover panel on the left of the window and hte sidebar will appear. Move your mouse away from the sidebar and it will disappear again.

Display Settings

The display options control what information is displayed for each tab in the sidebar. The following options are available:

&paneDisplay.content.favicon.label;:
This option controls the display of the website's icon in the display.

&paneDisplay.content.reload.label;:
Displays a reload button that when clicked reloads the tab.

&paneDisplay.content.throbber.label;:
Shows the loading throbber when the tab is currently loading.

&paneDisplay.content.security.label;:
If the tab is displaying a secure site then this option will display an indication of the security level.

&paneDisplay.content.close.label;:
Displays a button letting you close the tab quickly.

&paneDisplay.content.preview.label;:
Displays a thumbnail preview of the website displayed in the tab.

&paneDisplay.content.title.label;:
Shows the title of the website displayed in the tab.

&paneDisplay.content.uri.label;:
Shows the address of the address of the website displayed in the tab.

Preview Settings

&panePreviews.drawscrolled.label;:
When a web page has been scrolled down, this option will control whether the preview drawn is taken from what is seen in the browser window (the scrolled view) or from the area at the very start of the webpage.

&panePreviews.autorefresh.label;:
For pages that continually change it may be desirable to have the preview redraw continually. This option allows you to do this and lets you specify how often (in seconds) to draw the preview. The minimum time is every 5 seconds.

&panePreviews.showadvanced.label;:
Selecting this option will display some more advanced options that most casual users should not change.

Refresh the preview during page load:
If you have a fast computer you might want to have the page previews redraw during the page load. This option will let you enable this feature and specify how often (in milliseconds) to draw the preview. The minimum time is every 100 milliseconds. Note that this will slow down the loading of webpages and will have no effect on some websites.

&panePreviews.loaddelay.label;:
After a website has been loaded there is a slight delay before the preview is drawn to allow the page to complete drawing. This delay is initially set at 500 milliseconds but you may tweak it here if you wish. Setting the delay too low can cause unfinished pages to be displayed in the preview.
