使用 &tabsidebar.brand;

這份文件會解釋如何使用 &tabsidebar.brand;。

本節內容:
  • 打開和關閉 &tabsidebar.brand;
  • 使用縮圖預覽
  • 如何操作分頁

打開和關閉 &tabsidebar.brand;

要打開或關閉 Tab Sidebar，只要從 檢視 > 側邊欄 > &tabsidebar.menu.label; 切換就可以了。 想快速開關側邊欄也可以按鍵盤上 &accelKey;+&shiftKey;+T 或 &accelKey;+&shiftKey;+E。

使用縮圖預覽

使用 &tabsidebar.brand; 最大的好處就是可以預覽你所開啟的每個分頁畫面，當頁面重新載入時，縮圖預覽也會自動重繪。

如何操作分頁

&tabsidebar.brand; 和原來的分頁列一樣，提供了多種操作分頁的方法。下面是可使用的動作列表：

建立新分頁
在 Tab Sidebar 裡建立新分頁的方法和原來的分頁列一樣，只要在側邊欄中空白的地方按兩下即可。如果在分頁間的空白處按兩下的話，就會在兩個分頁間插入新分頁。 你也可以於按住 &ctrlKey; 後，再點側邊欄按滑鼠右鍵，接著選擇 &newTab.label;，就可以在最後面加入新分頁了。 將網址或書籤項目拖放到側邊欄中的空白區域（或分頁間的空白處），則會直接以該網址開啟新分頁。
關閉使用中或背景分頁
有幾個方法可以關閉分頁。最快的方法就是和原來的分頁列一樣，在側邊欄的分頁縮圖上按滑鼠中鍵關閉該分頁。也可以 按住 &ctrlKey; 後，再點側邊欄按滑鼠右鍵，接著選擇 &closeTab.label;。 不然如果你有勾選 Tab Sidebar 選項 顯示導覽按鈕的話，按一下關閉鈕也可以關閉分頁。
快顯功能表可以讓你關閉除了已選分頁外的其他分頁。只要於按住 &ctrlKey; 後，再點分頁按滑鼠右鍵 ，然後選擇 &closeOtherTabs.label; 即可。
重新載入使用中或背景分頁
和原來的分頁列相同，&tabsidebar.brand; 也可以於快顯功能表中重新載入單一或全部分頁。只要 按住 &ctrlKey; 後，再點側邊欄按滑鼠右鍵，然後選 &reloadTab.label; 或 &reloadAllTabs.label; 就可以了。預設的情況下，每個分頁縮圖上都會顯示重新載入的按鈕，所以也可以按該按鈕來重新載入分頁。 如果不想顯示按鈕，可以從 選項視窗 中關閉。
移動分頁順序
&tabsidebar.brand; 也可以像原來分頁列一樣，用拖放的方式移動分頁順序。先在分頁上按滑鼠左鍵，然後移動到你想要放置的地方後放開滑鼠。 在移動滑鼠時會出現箭頭指示，提示你放開滑鼠時分頁會移動到哪裡。
