Come usare &tabsidebar.brand;

Questo documento spiega come usare &tabsidebar.brand;

In questa sezione:
  • Visualizzare &tabsidebar.brand;
  • Usare le anteprime
  • Gestire le schede

Visualizzare &tabsidebar.brand;

Per visualizzare oppure chiudere la barra laterale si deve usare il menu Visualizza > Barra laterale > &tabsidebar.menu.label;. Alternativamente si può visualizzare oppure chiudere la barra laterale con la combinazione dei tasti &accelKey;+Maiusc+T oppure &accelKey;+Maiusc+E

Usare le anteprime

Certamente uno dei grandi benefici di usare &tabsidebar.brand; è il poter visualizzare le anteprime delle schede aperte. Le anteprime vengono automaticamente ridisegnate ogni volta che si carica una pagina

Gestire le schede

&tabsidebar.brand; si può usare in vari modi esattamente come la normale barra delle schede. Questa è una lista delle possibili azioni:

Creare nuove schede
Esattamente come nella normale barra delle schede, &tabsidebar.brand; permette di creare nuove schede nella barra laterale: con un semplice doppio clic del mouse in uno spazio vuoto della barra laterale, o anche fra le schede, verrà creata una nuova scheda. Alternativamente si può usare il clic con tasto destro del mouse premere il tasto &ctrlKey; e fare clic nella barra laterale e selezionare &newTab.label; per creare una nuova scheda in basso. È inoltre possibile trascinare e rilasciare l'indirizzo web di un sito, oppure un segnalibro in uno spazio vuoto della barra laterale, o anche fra le schede, per creare una nuova scheda e visualizzare tale indirizzo
Chiudere le schede attive ed in secondo piano
Ci sono alcune opzioni per chiudere le schede. Il metodo più rapido è lo stesso della barra principale delle schede: fare clic con il tasto centrale del mouse su una scheda nella barra laterale per chiuderla. Alternativamente si può usare il clic con tasto destro del mouse premere il tasto &ctrlKey; e fare clic nella barra laterale e selezionare &closeTab.label;. Infine con un semplice clic sul pulsante di chiusura, in caso si sia impostato nelle opzioni di visualizzazione della barra laterale di mostrare il pulsante di chiusura, verrà chiusa la scheda
Il menu contestuale permette di chiudere tutte le schede eccetto una. Fare clic con il tasto destro del mouse Premere &ctrlKey; e fare clic su una scheda per farla rimanere aperta e selezionare &closeOtherTabs.label;
Ricaricare le schede attive ed in secondo piano
Esattamente come la normale barra delle schede, &tabsidebar.brand; permette di ricaricare tramite menu contestuale una singola scheda, o tutte le schede: sarà sufficiente fare clic con il tasto destro del mouse premere il tasto &ctrlKey; e fare clic nella barra laterale e selezionare &reloadTab.label; oppure &reloadAllTabs.label;. Come impostazione predefinita ogni scheda contiene il pulsante 'Ricarica' per poter ricaricare le schede con un sol clic. Si può disattivare tale opzione nelle opzioni di visualizzazione
Spostare le schede
Esattamente come nella normale barra delle schede, &tabsidebar.brand; permette di spostare le schede nella barra laterale tramite trascinamento e rilascio. Tenere premuto su una scheda il primo tasto del mouse ed avviare il movimento del mouse per iniziare il trascinamento.
