&tabsidebar.brand; benutzen

Hier finden Sie Erläuterungen zur Nutzung der &tabsidebar.brand;.

In diesem Abschnitt:
  • &tabsidebar.brand; anzeigen und schließen
  • Vorschaubilder benutzen
  • Tabs steuern

&tabsidebar.brand; anzeigen und schließen

Um die Sidebar anzuzeigen oder zu schließen, benutzen Sie das Menü Ansicht > Sidebar > &tabsidebar.menu.label;. Alternativ ist auch ein schnelles Umschalten über die Tastaturkürzel &accelKey;+&shiftKey;+T oder &accelKey;+&shiftKey;+E möglich.
Eine dritte Möglichkeit bietet eine eigene Schaltfläche, die über Ansicht > Symbolleisten > Anpassen... angelegt werden kann.

Vorschaubilder benutzen

Einer der großen Vorzüge bei der Benutzung der &tabsidebar.brand; sind natürlich die Miniatur-Vorschaubilder für jeden einzelnen offenen Tab. Die Vorschaubilder werden bei jeder Änderung einer Seite automatisch neu gezeichnet.

Tabs steuern

&tabsidebar.brand; bietet die gleichen Möglichkeiten, mit Tabs zu arbeiten, wie die normale Tab-Leiste auch. Sehen sie im Folgenden eine Liste der verfügbaren Aktionen:

Neue Tabs erstellen
Neue Tabs können in der Sidebar auf die gleiche Weise erstellt werden wie in der Tab-Leiste. Durch Doppelklick auf eine freie Stelle der Sidebar oder zwischen den Tabs wird dort ein neuer Tab geöffnet. Alternativ können Sie auch durch Rechtsklick &ctrlKey;+Klick auf die Sidebar den Eintrag &newTab.label; im Kontextmenü auswählen. Des weiteren ist es möglich, mit der Maus eine Website-Adresse oder ein Lesezeichen auf eine beliebige freie Stelle der Sidebar zu ziehen, um die zugehörige Seite in einem neuen Tab anzeigen zu lassen.
Aktive Tabs und Tabs im Hintergrund schließen
Es gibt verschiedene Möglichkeiten zum Schließen von Tabs. Der schnellste Weg ist, genau wie in der Tab-Leiste, der Mittelklick auf einen Tab (Vorschau) in der Sidebar. Alternativ können Sie auch durch Rechtsklick&ctrlKey;+Klick auf die Sidebar den Eintrag &closeTab.label; im Kontextmenü auswählen. Letztendlich können Sie auch die in jedem Tab vorhandene Schließen-Schaltfläche benutzen.
Über das Kontextmenü ist es auch möglich, alle Tabs außer einem zu schließen. Durch einen Rechtsklick &ctrlKey;+Klick auf den Tab, der offen bleiben soll, können Sie den Eintrag &closeOtherTabs.label; auswählen.
Aktive Tabs und Tabs im Hintergrund neu laden
Wie die Tab-Leiste selbst bietet auch &tabsidebar.brand; die Möglichkeit, einzelne oder alle Tabs über das Kontextmenü neu zu laden. Führen Sie dazu einfach einen Rechtsklick&ctrlKey;+Klick auf die Sidebar aus und wählen Sie &reloadTab.label; oder &reloadAllTabs.label;. Standardmäßig enthält jeder Tab eine eigene Schaltfläche, um ihn mit einem einzigen Klick neu laden zu können. Diese lässt sich aber auch über die Einstellungen für die &tabsidebar.brand; deaktivieren.
Tabs verschieben
Wie die Tab-Leiste erlaubt auch &tabsidebar.brand; das freie Verschieben von Tabs durch einfaches Ziehen mit der Maus. Halten Sie dazu einfach die linke Maustaste auf einem Tab gedrückt and beginnen Sie, die Maus zu bewegen, um das Verschieben zu starten. Wenn Sie nun die Maus innerhalb der Sidebar auf und ab bewegen, erscheint ein kleiner Pfeil, der anzeigt, wo der Tab platziert wird, wenn sie die Taste wieder loslassen.

Hinweis: Bei gleichzeitigem Einsatz anderer Tab-Erweiterungen, die zusätzliche Aktionen bieten (wie bspw. Tab Mix Plus) stehen diese ebenfalls in der Tab Sidebar zur Verfügung.
